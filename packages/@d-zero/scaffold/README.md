# D-ZERO フロントエンド標準開発環境

詳細は[ガイドライン](https://guidelines.d-zero.co.jp)を参照してください。

## コマンド

| コマンド     | 実行される内容                                       |
| ------------ | ---------------------------------------------------- |
| `yarn`       | 必要なパッケージのインストール                       |
| `yarn dev`   | 開発用ローカル環境の起動（ http://localhost:8080/ ） |
| `yarn lint`  | リントチェック                                       |
| `yarn build` | ビルド                                               |
| `yarn up`    | 依存パッケージのアップデート                         |
